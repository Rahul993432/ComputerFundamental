Computer Forth Generation :-

In this processor was used as a CPU. The central processing unit build processor. Memory, higher processor and other supporting chips also uses IC (LSI and VLSI ) in earlier generation computer cache memory, MMU and FTU uses seprate IC but with the development of VLSI technology it become possible to integrate all these component on single IC along with CPU.

The importent features forth generation computer are :-

  • Micro processor is used as a CPU.
  • Processing speed is increased to pico speed.
  • CRT screen was developed.
  • Lesser printer, Inkjet printer and scaner were developed.
  • Compter are more user frindly which uses many high level language for programming.

Example of forth generation of computer are :-

Honey well 6080 series, Motorola's 68000, 68020, 68030

Intel's 8056, 80286

CRAY-1, CRAY-2, CRAY-XMD.
